# VoiceFlow | Real-time English-Finnish Translator

VoiceFlow is a sleek, web-based voice-to-text translation application that enables seamless communication between English and Finnish speakers. It leverages modern web technologies to provide an interactive and responsive experience with minimal latency.

## 🚀 Features

- **Live Voice Recognition**: Instant speech-to-text using the Web Speech API.
- **Bi-directional Translation**: Easily switch between English -> Finnish and Finnish -> English with a toggle.
- **Real-time Feedback**: Displays interim results as you speak for a "zero-latency" feel.
- **Premium UI**: A modern dark-themed interface with glassmorphism effects and smooth animations.
- **Cross-compatible**: Works on modern Chromium-based browsers (Chrome, Edge).

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Premium styling with glassmorphism, flexbox, and CSS animations.
- **JavaScript**: Core logic for speech recognition and API integration.
- **Web Speech API**: Real-time voice processing.
- **MyMemory API**: Reliable translation engine.

## 📖 How to Use

1.  **Open the App**: Launch `index.html` in a modern browser like Chrome or Edge.
2.  **Select Language**: Use the toggle button at the top to choose your translation direction:
    - **English to Finnish** (Toggle Off)
    - **Finnish to English** (Toggle On)
3.  **Start Listening**: Click the **"Start Listening"** button.
4.  **Speak**: Grant microphone permission if prompted. Begin speaking naturally.
5.  **View Results**: Your recognized speech will appear in the top card, and its translation will follow in the bottom card.
6.  **Stop**: Click **"Recording..."** to stop the recognition.

## ⚠️ Requirements

- An active internet connection (for translation API).
- A modern browser that supports the Web Speech API (Chrome v25+ or Edge v79+ recommended).
- Microphone access permissions.
