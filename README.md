# Voice Translator (English <-> Finnish)

A premium, real-time voice translator web application built with modern web technologies. This app allows users to speak in English or Finnish and get instantaneous text translations.

## 🚀 Features

- **Live Voice Input**: Uses the Web Speech API for real-time speech recognition.
- **Bi-directional Translation**: Toggle seamlessly between English-to-Finnish and Finnish-to-English.
- **Micro-Animations**: Features a dynamic, "alive" UI with background blobs and hover effects.
- **Glassmorphism Design**: A sleek, dark-themed interface with frosted glass effects and premium typography.
- **Zero Latency Feel**: Displays interim speech results as you talk for immediate feedback.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Advanced styling, gradients, and animations.
- **JavaScript (ES6+)**: Core logic for speech-to-text and translation.
- **Web Speech API**: Browser-native speech recognition.
- **MyMemory API**: Reliable translation service.

## 🚦 Getting Started

### Prerequisites

- A modern browser (Google Chrome or Microsoft Edge are recommended for Web Speech API support).
- Microphone access granted to the site.

### Installation / Running Locally

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ashishdhungana000-cell/Translation_APP.git
    ```
2.  **Open the project**:
    - Simply open `index.html` in your browser.
    - Or use a local server like `npx serve`:
      ```bash
      npx serve .
      ```

## 📖 How to Use

1.  **Select Direction**: Use the toggle switch to choose between "English to Finnish" or "Finnish to English".
2.  **Start Listening**: Click the **🎤 Start Listening** button.
3.  **Speak**: Talk into your microphone. You will see your voice appearing in the "Voice Input" area.
4.  **Translate**: Once you stop speaking, the translation will automatically appear in the "Translation" area.
5.  **Stop**: Click the button again to stop the microphone.

## 🎨 Design Philosophy

The application utilizes a **Dark Mode** aesthetic with **HSL tailored colors** to ensure a premium look. The background features animated blobs that move subtly, creating a sense of depth and interactivity.
